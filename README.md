# appX

## If running for the first time
#### • Navigate to main folder
#### • npm install
#### • bower install

## To run on ios
#### • ionic build ios
#### • ionic emulate ios

## To run on android
#### • ionic build android
#### • ionic emulate android

## To run on the web or mobild ios/android
#### • ionic serve --lab (runs in the web for ios/android)
#### • ionic serve (runs a responsive web build)
